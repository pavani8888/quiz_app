# Quiz App

A simple and interactive Quiz Application built using HTML, CSS, and JavaScript. The app allows users to answer multiple-choice questions, navigate between questions, submit the quiz, view their score, and review correct answers after completion.

## Features

- Multiple-choice questions
- Previous and Next navigation
- Score calculation
- Answer validation
- Show correct and incorrect answers
- Reload quiz functionality
- Clean and responsive user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Project Files

- `index.html` – Structure of the quiz application
- `style.css` – Styling and layout
- `script.js` – Quiz logic and functionality

## How to Run

1. Download all project files.
2. Keep `index.html`, `style.css`, and `script.js` in the same folder.
3. Open `index.html` in any modern web browser.

## How It Works

1. Select an answer for each question.
2. Click **Next** to move to the next question.
3. Use **Previous** to revisit earlier questions.
4. Click **Submit** on the last question.
5. View your final score.
6. Click **Show Answers** to review the correct answers.
7. Use **Reload** to restart the quiz.
